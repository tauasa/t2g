# T2G

T2G is a HATEOAS REST service for tracking golf scores. Built with Spring Boot and Spring JPA.

### Golf Score Tracker (WIP)
Tracks the following data points for each hole:
* Strokes
* Fairways
* Drive Distance
* GIR
* Putts
* Penalties
* Sandies
* Mulligans
