# T2G

T2G is a REST service for tracking golf scores. Built with Spring Boot and Spring JPA.

### Golf Score Tracker
Tracks the following data points:
* Strokes
* Fairways
* Drive Distance
* GIR
* Putts
* Penalties
* Sandies
* Mulligans
