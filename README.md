# T2G (Tee-2-Green)

A HATEOAS REST service for tracking golf scores. Built with Spring Boot and Spring JPA.

### Golf Score Tracker
Tracks the following data points for each hole:
* Strokes
* Fairways
* Drive Distance
* GIR
* Putts
* Penalties
* Sandies
* Mulligans
